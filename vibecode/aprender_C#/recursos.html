C# Cheat Sheet

Estrutura (Top-level)

using System;

Console.WriteLine("Hello C#!");

void Ola() => Console.WriteLine("Oi");
Ola();

Tipos e Variáveis

var x = 10;           // inferência
string s = "Texto";
double d = 10.5;
bool b = true;
decimal m = 5.99m;    // dinheiro

Interpolação de String

string nome = "Ana";
int idade = 25;
// Use $ no início
Console.WriteLine($"Oi {nome}, {idade} anos");
Console.WriteLine($"{valor:F2}"); // 2 casas

Coleções (List)

var lista = new List<string>();
lista.Add("Item 1");
lista.Remove("Item 1");
int count = lista.Count;

LINQ (Consultas)

using System.Linq;

var filtrados = lista
    .Where(x => x.Length > 5)
    .OrderBy(x => x)
    .ToList();

Classes e Props

}

if (b) { } else { }

    case 1 => "Um",
    _ => "Outro"
}; // switch expression

Loops

foreach (var item in lista) { }
for (int i=0; i<10; i++) { }

Tratamento de Erros

try {
    // código perigoso
} catch (Exception ex) {
    Console.WriteLine(ex.Message);
} finally {
    // sempre roda
}

Async/Await

async Task<string> GetData() {
    await Task.Delay(1000);
    return "Pronto";
}

Erros Comuns em C#

❌ NullReferenceException
Tentativa de acessar algo que está null. Use if (obj != null) ou obj?.Member.
❌ Namespace Errado
Esqueceu o using necessário (ex: using System.Linq; para filtrar listas).
❌ Atribuir vs Comparar
Assim como em C, = atribui e == compara. O C# avisa mais, mas cuidado.
❌ Async sem Await
O código continua rodando sem esperar a Task terminar. Use await!
❌ Erro de Ponto Flutuante
Usar double para dinheiro. Sempre use decimal (sufixo m) para valores monetários.
❌ Static vs Instance
Tentar acessar um membro não-estático diretamente pela classe sem criar um objeto (new).
❌ Esquecer ponto e vírgula
Mesmo em C# moderno, o ; é obrigatório na maioria dos comandos.
❌ Indice fora do Array
Indices começam em 0. Se o Count é 5, o último índice é 4.
